Space.

Overview:
Space. is a demo Web site project that was originally built in August and abandoned. The project aim was to imitate the style and layout of third party Apple Store websites.

Setup Instructions:
Download the file space.zip.
Extract (unzip) the contents.
Open the main file yonyll.html in your browser.

Notes:
 All the design and construction were done by me.
 The JavaScript in the project was not by me.

Status:
This is an unfinished project and is being used as a demonstration and not a finished product.
